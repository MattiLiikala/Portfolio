Projects

Hamster shelter is a game project in which I work as a graphic designer and programmer. The game is at the moment on early beta version. The idea of the game is simply to build a shelter for the hamsters before the meteor shower begins. Our team was nominated as the best game in the ICT ShowRoom -event on 10 march 2016.

Under the great sky is a game project I have participated as a graphic designer. It is a point and click mobile game based on a scientist named Pjotr Mulkov. In a nutshell the main goal is to save the galaxy. I did the ui components and some of the 3D modelling. the game is not yet published, it's on early alfa testing at the moment.
